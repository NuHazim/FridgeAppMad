# 🧊 FridgeBuddy

> **WIA2007 — Mobile App Development Project**  
> A smart Android app that helps you manage your fridge, reduce food waste, and cook smarter.

---

## 📖 About

**FridgeBuddy** is a smart fridge management app built for Android (Java). It tracks your ingredients, alerts you before items expire, suggests AI-powered recipes from what you already have, and syncs your shopping list directly into your inventory — so nothing gets wasted and nothing gets forgotten.

---

## ✨ Features

### 🔐 Authentication
- Login and registration with email/password
- Social login via **Google**, **Facebook**, or **Apple**

### 📦 Inventory Page
- Manually add ingredients like a simple to-do list
- **Scan barcodes** to auto-fill item details using the [Open Food Facts API](https://world.openfoodfacts.org/)
- Set and track **expiration dates** for each item
- Receive **reminders** when items are approaching their expiry date

### 🛒 Shopping List Page
- A clean to-do list for items you need to buy
- Check off items as you shop
- Once done, tap **"Complete Purchase"** — all checked items are automatically transferred to your **Inventory** with details pre-filled

### 🍳 Recipes Page (AI-Powered)
- Recipes are suggested by **OpenAI** based on the ingredients currently in your inventory
- Each recipe displays:
  - Food name
  - Required ingredients
  - Step-by-step cooking instructions
- Save any recipe to your **Favourites** for quick access later

### 👤 Profile Page
- View and manage your personal profile information

## 🚀 Getting Started

### Prerequisites
- Android Studio (latest stable)
- Android SDK (API 24+)
- Java 8 or higher

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/NuHazim/FridgeAppMad.git
   ```
2. Open the project in **Android Studio**
3. Let Gradle sync and resolve dependencies
4. Run the app on an emulator or physical device
